# Babel 是什么，他是如何工作的

## Babel 的概念

Babel 最本质上是一个 JavaScript 编译器(Compiler)，它能够将最新的语法转换成低版本的浏览器可以识别的代码

## Babel 有哪些模块

- babel-cli
- babel-core
- babel-runtime
- babel-node
- babel-polyfill
...

## Babel 编译的过程

Babel 总共分为三个阶段：解析、转换、生成
