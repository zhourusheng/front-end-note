## TypeScript 常用语法

### 1. 基础类型
  * boolean
    * `let isDone: boolean = false`
  
  * number
    * `let decLiteral: number = 20`
  
  * string
    * `let name: string = 'bob'`
  
  * array
    * `let list: number[] = [1, 2, 3]`
    * `let list: Array<number> = [1, 2, 3]`
  
  * 元组 Tuple
  
  * 枚举 enum
  
  * any
  
  * void
